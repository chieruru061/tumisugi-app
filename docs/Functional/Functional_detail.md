## 機能名
- 日本語名: お積みもの詳細
- 英語名 : Tumi Detail

## 概要
お積みものの詳細を表示、編集するための画面

## トリガー
- お積みもの一覧画面.詳細ボタンクリック

## 機能一覧
※DDL: ドロップダウンリスト

### 初期表示
1. controllerは画面情報を返却する
    1. カテゴリDDLに表示する、カテゴリ一覧
    2. 積み状態DDLに表示する、ステータス一覧
    3. プラットフォームDDLに表示する、プラットフォーム一覧
    4. 一覧画面で詳細ボタンを押したアイテムに紐づく詳細情報
2. Viewは受け取った情報をDDLに設定
3. Viewは受け取ったお積みもの詳細情報を各表示項目に設定

### 更新ボタン
1. 入力バリデーションを実施する
2. バリデーションエラーがない場合、controllerにフォーム情報を渡す
3. controllerはお積みもの情報をDBに登録する
4. 削除成功後、お積みもの一覧画面に遷移する

### 戻るボタン
1. お積みもの一覧画面に遷移する

### その他

## 表示項目（一覧）
- 画面名: お積みもの詳細
- タイトルラベル
- カテゴリラベル
- お積みもの状態ラベル
- プラットフォームラベル
- メモラベル
- 削除ボタン
- 戻るボタン

## API / コントローラ設計
※条件入力が空ならクエリ文字列に含めない。
- Controller: Index()
  - GET /Add/Index?id={tumiId}
  - Response: RazorView
- Controller: Delete()
  - POST /Add/Delete
  
  body={tumiId}
  - Response: RazorView
