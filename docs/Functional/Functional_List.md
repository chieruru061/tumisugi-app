## 機能名
- 日本語名: お積みもの一覧
- 英語名 : Tumi List

## 概要
お積みもの一覧を表示する画面

## トリガー
- リンクアクセスのみ。ホーム画面となる。

## 機能一覧
※DDL: ドロップダウンリスト

※お積みもの一覧のデフォルト表示順は積み状態の昇順, 更新日の降順とする。

### 初期表示
1. TumiSugiにアクセスする
2. 初期表示でcontrollerは画面情報を返却する
    1. カテゴリDDLに設定する、カテゴリ一覧
    2. 積み状態DDLに設定する、ステータス一覧
    3. プラットフォームDDLに設定する、プラットフォーム一覧
    4. 一覧表示用のお積みもの一覧情報
3. Viewは受け取った情報をDDLに設定
4. Viewは受け取ったお積みもの一覧情報を10件ずつ表示する(詳細は検索機能参照)


### 検索
1. ユーザは検索条件を入力し、検索を行うことができる。
2. 検索条件
    1. カテゴリDDL
    2. 積み状態DDL
    3. プラットフォームDDL
    4. タイトルテキストボックス
3. 検索ボタン押下で、controllerにパラメータとして検索条件を渡す
4. controllerは検索条件に合致するお積みもの一覧情報を返却する
5. Viewは受け取ったお積みもの一覧情報を10件ずつ表示する

### 追加ボタン
1. 追加ボタン(+)クリックで、お積みもの追加画面に遷移する。
2. パラメータ
    1. なし

### 詳細ボタン
1. 各お積みものの詳細リンククリックで、お積みもの詳細画面に遷移する。
2. パラメータ
    1. お積みもの情報ID

### 削除ボタン
1. 各お積みものの削除リンククリックで、お積みもの削除画面に遷移する。
2. パラメータ
    1. お積みもの情報ID

### その他
1. ユーザはお積みもの一覧の各列をクリックでソートができる。昇順/降順の切り替えも可能。下記に、ソート順を定義する。クリック列をプライマリ、更新日をセカンダリとする。
    1. クリックしたソート列
    2. 更新日の昇順
2. ページャーで前の10件、次の10件に遷移できる。(対象がない場合は非活性)

## 代替フロー（エラーや分岐）
- A1: データが0件の場合 → 何も表示しない
- A2: DB 接続エラー → エラーページまたは通知を表示（ログ出力）

## API / コントローラ設計
※条件入力が空ならクエリ文字列に含めない。
- Controller: Index
  - GET /List/Index?page={1}&pageSize={10}&sort={4:1,5:2}
  - Response: RazorView
- Controller: Search
  - GET /List/Search?page={ページャ番号}&pageSize={10}&sort={ソート対象を表す数値:1or2,4:2}&title={検索テキストボックス}&genre={カテゴリDDLの値}&status={積み状態DDLの値}&platform={プラットフォームDDLの値}
  - Response: RazorView
