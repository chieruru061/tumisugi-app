## 機能名
- 日本語名: お積みもの追加
- 英語名 : Tumi Add

## 概要
お積みものを新規追加するための画面

## トリガー
- お積みもの一覧画面.追加ボタンクリック

## 機能一覧
※DDL: ドロップダウンリスト

### 初期表示
1. controllerは画面情報を返却する
    1. カテゴリDDLに表示する、カテゴリ一覧
    2. 積み状態DDLに表示する、ステータス一覧
    3. プラットフォームDDLに表示する、プラットフォーム一覧
2. Viewは受け取った情報をDDLに設定

### 追加ボタン
1. 入力バリデーションを実施する
2. バリデーションエラーがない場合、controllerにフォーム情報を渡す
3. controllerはお積みもの情報をDBに登録する
4. 登録成功後、お積みもの一覧画面に遷移する

### 戻るボタン
1. お積みもの一覧画面に遷移する

### その他

## 表示項目（一覧）
- 画面名: お積みもの追加
- タイトルテキストボックス
- カテゴリDDL
- お積みもの状態DDL
- プラットフォームDDL
- メモテキストボックス
- 追加ボタン
- 戻るボタン

## バリデーション
  - 長さチェック
  - 禁止文字チェック(例: SQLインジェクション対策,XSS対策)
  - 空白・全角スペースの正規化
  - 文字種の正規化(例: 全角→半角)

## API / コントローラ設計
※条件入力が空ならクエリ文字列に含めない。
- Controller: Index()
  - GET /Add/Index
  - Response: RazorView
- Controller: Create()
  - POST /Add/Create
  
  body={title,genre,status,platform,memo}
  - Response: RazorView
