## 機能名
- 日本語名: お積みもの削除
- 英語名 : Tumi Delete

## 概要
お積みもののを削除するための画面

## トリガー
- お積みもの一覧画面.削除ボタンクリック

## 機能一覧
※DDL: ドロップダウンリスト

### 初期表示
1. controllerは画面情報を返却する
    1. 一覧画面で削除ボタンを押したアイテムに紐づく詳細情報
2. Viewは受け取ったお積みもの詳細情報を各表示項目に設定

### 削除ボタン
1. 入力バリデーションを実施する
2. バリデーションエラーがない場合、controllerにフォーム情報を渡す
3. controllerはお積みもの情報をDBに登録する
※画面遷移は行わない

### 戻るボタン
1. お積みもの一覧画面に遷移する

### その他

## 表示項目（一覧）

## バリデーション
  - 長さチェック
  - 禁止文字チェック(例: SQLインジェクション対策,XSS対策)
  - 空白・全角スペースの正規化
  - 文字種の正規化(例: 全角→半角)

## API / コントローラ設計
※条件入力が空ならクエリ文字列に含めない。

  - Response: RazorView
- Controller: Update()

  body={title,genre,status,platform,memo}
